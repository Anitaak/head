This is a quick set of notes for generating server skeletons and
client stub code from the MifosService WSDL.

Assume:

AXIS2_HOME is set to point to your AXIS2 Install
MIFOS_HOME is set to the top of the main mifos application source tree
MIFOSSOAP_HOME is set to this directory

Execute the following:

$ cd $AXIS2_HOME/bin

$ ./WSDL2Java.sh -uri $MIFOS_HOME/src/org/mifos/api/resources/MifosService.wsdl \
    -o $MIFOS_HOME -p org.mifos.api -ss -ssi -sd

$ ./WSDL2Java.sh -uri $MIFOS_HOME/src/org/mifos/api/resources/MifosService.wsdl \
    -o $MIFOSSOAP_HOME -p org.mifos.api -t

The first WSDL2Java command generates the server-side skeletons. These
need to be customized.

The second WSDL2Java command generates the client stubs. Include these
in your program. Check out the unit tests for an example.
